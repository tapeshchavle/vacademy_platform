🎨 Payment Dialog UI Test Suite

This is a static HTML page to help you access the payment dialog test suite.

📋 How to Access the Test Suite

  1. Make sure your dev server is running: npm run dev
  2. Navigate to: http://localhost:5173/payment-dialog-test
  3. If that doesn't work, try: http://localhost:5173/payment-dialog-test/
  4. Alternative: Import SimplePaymentDialogTest component in any existing page
